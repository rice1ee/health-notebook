# 健康记录本 🩺

给老年人用的血压 / 血糖 / 体重记录小应用。**无需安装、无需联网、数据不上传**，
用浏览器打开 `index.html` 就能用；也支持像手机 App 一样"添加到主屏幕"离线使用（PWA）。

## 怎么用

1. 打开 `index.html`（推荐 Chrome / Edge / Safari）。
2. 先到 **我的** 页填一次个人信息：出生年份（用于按年龄评判）、身高（用于体重 BMI 评判）、姓名（印在打印表格上）。
3. 每天在 **记录** 页填入血压等数值，点"保存这条记录"。
   - 输入血压、血糖时会实时提示是否正常。
4. **历史** 页按日期查看记录，每条自动给出评判标签（正常 / 血压偏高 / 血糖偏高 等）。
5. **趋势** 页查看血压、体重折线图，带 140/90 警戒虚线。
6. **打印** 页：
   - **打印记录表**：最近 7/30/90 天记录 + 姓名年龄 + 每条评估，就诊时给医生看；
   - **打印空白模板**：手写记录表（每页一周，早晚各一行），自动填好姓名；
   - **导出 / 导入 CSV**：备份数据，可用 Excel 打开，换设备后导入恢复。

## 在手机上使用

**方式一（推荐）：部署后"添加到主屏幕"，像 App 一样用**

1. 把 `健康记录` 文件夹发布到任意免费静态托管（GitHub Pages、Gitee Pages、Netlify、Cloudflare Pages 均可，需 https）。
2. 用手机浏览器打开网址：
   - 安卓 Chrome/Edge：菜单 → "添加到主屏幕"（或"安装应用"）；
   - iPhone Safari：分享按钮 → "添加到主屏幕"。
3. 之后从主屏幕图标打开，全屏无地址栏，**断网也能用**（Service Worker 已缓存全部页面）。

**方式二：直接拷文件到手机**

把整个文件夹传到手机（微信/网盘/数据线），用文件管理器以浏览器打开 `index.html`。安卓可行；iPhone 对本地文件支持差，建议用方式一。

> 注意：数据保存在"当时打开它的那个浏览器"里，电脑和手机的数据互不相通，可用 CSV 导出/导入互相搬运。

## 健康评判标准（按年龄自动调整）

| 指标 | 标准 |
|------|------|
| 血压 | 正常 <120/80；正常高值 120–139/80–89；高血压 ≥140/90；**80 岁及以上控制目标放宽到 150/90 以内**（140–149/<90 判"略高"而非"高血压"）；<90/60 提示偏低 |
| 心率 | 正常 60–100 次/分，<50 或 >120 建议就医 |
| 血糖 | 空腹 3.9–6.1 正常，≥7.0 达糖尿病标准；餐后 2h <7.8 正常，≥11.1 达糖尿病标准；<3.9 低血糖提醒 |
| 体重 | 填写身高后按 BMI 评判：偏瘦 <18.5，正常 18.5–24，超重 24–28，肥胖 ≥28 |

参考国内常用指南，仅供日常参考，请遵医嘱。

## 适老化设计

- 全局字号 ≥ 20px，输入数字 24px，按钮高度 ≥ 56px；
- 每个标签页只做一件事；时段按当前时间自动预选；
- 高对比暖色配色，评判用绿（正常）/ 橙（注意）/ 红（偏高）直观标色。

## 技术说明

- 纯 HTML + CSS + JavaScript，零依赖，离线可用；
- 数据保存在浏览器 `localStorage`（记录：`health_records_v1`，个人信息：`health_profile_v1`）；
- 趋势图为自绘 SVG；打印通过 `@media print` 实现 A4 排版；
- PWA：`manifest.webmanifest` + `sw.js`（缓存优先），需 https（或 localhost）才会注册。

## 文件结构

```
健康记录/
├── index.html            # 页面结构（五个标签页 + 打印区域）
├── css/style.css         # 适老化样式 + 打印样式
├── js/app.js             # 存取、评判、图表、打印、CSV 逻辑
├── manifest.webmanifest  # PWA 清单（名称、图标、主题色）
├── sw.js                 # Service Worker（离线缓存）
├── icon-192.png / icon-512.png  # 应用图标
└── README.md
```

## 注意事项

- 数据跟随"浏览器 + 网址（或文件路径）"，请勿随意改动部署地址或移动文件夹；
- 清除浏览器缓存/网站数据会删除记录，建议定期"导出 CSV"备份。
