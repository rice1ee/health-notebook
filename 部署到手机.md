# 怎么把"健康记录本"装到安卓手机上

## 方式一：GitHub Pages（推荐，免费、有 https、可离线）

原理：把这个文件夹发布成一个网址，手机浏览器打开后"添加到主屏幕"，
它就会像普通 App 一样出现在桌面上，有图标、全屏、断网也能用。

**电脑上（一次性操作）：**

```bash
cd 健康记录
git init && git add -A && git commit -m "健康记录本"
gh repo create health-notebook --public --source=. --push
# 开启 Pages（main 分支根目录）
gh api repos/{owner}/health-notebook/pages -X POST \
  -f "source[branch]=main" -f "source[path]=/"
```

等 1–2 分钟，网址就是：`https://<你的用户名>.github.io/health-notebook/`

**手机上（教长辈操作，只需一次）：**

1. 用微信把网址发到手机，点开后选择"在浏览器打开"（推荐 Chrome 或系统浏览器）；
2. 点浏览器菜单（右上角三个点）→ **"添加到主屏幕"** 或 **"安装应用"**；
3. 回到手机桌面，会多一个红色心形图标 **"健康记录"**，以后点它就行；
4. 第一次打开时到"我的"页开启提醒，手机会询问"是否允许通知"，点**允许**。

> 以后更新功能：电脑上改完代码 `git add -A && git commit -m "更新" && git push`，
> 手机上关掉应用重新打开一两次就是新版了。

## 方式二：不上网，直接拷文件（应急用）

1. 把整个 `健康记录` 文件夹压缩发到手机（微信/QQ/数据线）；
2. 手机上解压，用文件管理器点 `index.html`，选择用浏览器打开。

缺点：没有桌面图标、不能弹通知、每次要找文件打开，只适合临时看。

## 方式三：家里局域网（有 NAS 或旧电脑常开的话）

把文件夹放进 NAS 的 Web 目录，手机连家里 WiFi 访问。
注意：非 https 地址无法"安装到桌面"和弹通知，体验不如方式一。

---

## 提醒功能说明（重要）

网页应用受浏览器限制：**应用完全关闭时无法定时弹通知**。实际效果是——

| 场景 | 能否提醒 |
|------|---------|
| 应用开着 / 挂在后台 | ✅ 到点弹通知（该时段已记录则不打扰） |
| 打开应用时发现早/晚还没记录 | ✅ 顶部显示橙色横幅，点击直达记录页 |
| 应用完全没打开 | ❌ 弹不了 |

**给长辈的最稳做法**：手机自带"时钟"App 里设两个每天重复的闹钟
（如 7:00"量血压"、19:00"量血压"），闹钟一响 → 量血压 → 点桌面红心图标记一笔。
应用内提醒作为第二道保险。
